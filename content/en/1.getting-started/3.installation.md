# Installation


Exnest AI SDK provides a unified interface to access various AI models easily in your Node.js project.

## Prerequisites

- Node.js >= 16.x
- npm >= 7.x

## Install the SDK

Run the following command in your project directory:

```bash
npm install @exnest-dev/ai
```
